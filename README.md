# Projeto IDEP: Python + Power BI
O que foi feito em Python: Limpeza dos dados:  Remoção de valores nulos nas colunas principais (IDEP_FINAL e NSE)  Conversão de tipos (texto para número)  Criação de novas colunas:  evolucao_IDEP (diferença entre IDEP final e inicial)  crescimento_percentual (evolução em porcentagem)  categoria_evolucao (classificação: Avanço significativo, Estabilidade, Declínio, etc.)  Exportação: Bases tratadas salvas em Excel para uso no Power BI.  O que foi feito no Power BI: Criação de um dashboard interativo com gráficos e filtros  Visualização das melhores e piores escolas por região  Análise da evolução do IDEP entre 2023 e 2024
